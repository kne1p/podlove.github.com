Creating a Podcast Archive Page

Would you like to have a page listing all your podcast episodes like newz-of-the-world.com/archive? This is a planned feature for the Publisher but until it actually ships, you can still create such a page with the help of another plugin.

Steps

  1. Install the Archivist plugin.
  2. Go to Settings > Archivist and add a template named “podcast” (or whatever you like).
    Copy & paste the template below or build your own.
  3. Create a WordPress page called “Archive” and paste the following shortcode: [archivist template="podcast" query="post_type=podcast"]. Done!
